## ✨ ¿Por Qué Elegir Zuno?

- ⚡ <strong>Extremadamente Rápidas</strong>  
  Inspiradas por Asio standalone, Zuno entrega rendimiento asincrónico sin sobrecarga que deja en el polvo a las tradicionales frameworks.

- 🔐 <strong>Seguras por Diseño</strong>  
  El soporte para TLS está integrado desde la base. Sin trucos. Sin envoltorios. Solo seguridad limpia y nativa.

- 🧩 <strong>API Expresiva</strong>  
  Inspiradas en Express.js, Zuno te da una interfaz limpia e intuitiva — pero en modern C++.

- 🛠️ <strong>Módulas y Escalables</strong>  
  Arquitectura tu aplicación con módulos reutilizables y escala fácilmente conforme crecen tus necesidades.

- 📦 <strong>Ligera y Portátil</strong>  
  Sin dependencias pesadas. Solo tú, Asio y la carretera abierta.

## 🧠 Construido para Constructores

Sea que estés construyendo microservicios, sistemas embebidos o aplicaciones web completas, Zuno te da las herramientas para mover rápido y construir con confianza.

export const code = `
#include <zuno/zuno.hpp>

void main(){
  zuno::App app;

  app.get("/", [](auto req, auto res) {
    res.send("¡Hola, mundo desde Zuno!");
  });

  app.listen(3456);
}
`;
