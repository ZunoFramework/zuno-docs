# 🤝 Contribuyendo con Zuno

Zuno es un proyecto abierto y en constante crecimiento. Ya sea que estés fijando un error de ortografía, mejorando la documentación o construyendo una nueva extensión—tu contribución importa.

Bienvenidos a desarrolladores de todos los niveles de experiencia. Si estás emocionado por el moderno C++ y las API web limpias, has llegado al lugar correcto.
---

## 🚀 Comenzando
1. **Fork el repositorio**
   Ve a [github.com/ZunoFramework/zuno](https://github.com/ZunoFramework/zuno) y haz clic en “Fork”.

2. **Clona tu fork**

<Code
  lang="bash"
  code={`git clone https://github.com/your-username/zuno.git
cd zuno`}
/>

3. **Crea una nueva rama**

<Code
  lang="bash"
  code={`git checkout -b feature/mi-idea-genial`}
/>

4. **Compila el proyecto**

<Code
  lang="bash"
  code={`cmake -B build 
  cmake --build build`} />

5. **Realiza tus cambios**
   Sigue la estilo de codificación y mantén los commits enfocados.

6. **Empuja y abre una PR**

<Code lang="bash"
  code={`git push origin feature/mi-idea-genial`}/>

Luego, abre una solicitud de PR en GitHub.
---

## 🧠 ¿Qué Puedo Contribuir?

- 📚 **Documentación** – tutoriales, ejemplos, correcciones de ortografía
- 🧩 **Extensiones** – nuevas middleware, utilidades o integraciones
- 🐞 **Correcciones de errores** – agrupa problemas o mejora la gestión de errores
- ⚙️ **Mejoras en el núcleo** – rendimiento, TLS, routing
- 💡 **Ideas** – abre una discusión o sugiere una característica
---

## 🧭 Guías

- Mantén las PR enfocadas y atómicas
- Escribe mensajes de commit claros
- Agrega pruebas si es aplicable
- Documenta nuevas características en el archivo `.mdx` adecuado
- Sé respetuoso y colaborativo
---

## 🏷️ Etiquetas de problemas

| Etiqueta | Significado |
|----------|-------------|
| `good first issue` | Ideal para nuevos usuarios |
| `bug`     | Problema confirmado o retroceso |
| `enhancement` | Solicitud de característica o mejora |
| `discussion`  | Idea abierta o pregunta sobre diseño |
| `docs`        | Contribución relacionada con la documentación |
---

## 📬 Preguntas o Ideas?

- Abre una [discusión en GitHub](https://github.com/ZunoFramework/zuno/discussions)
- Inicia un nuevo problema con la etiqueta `discussion`
- Contacta al chat de comunidad (próximamente)
